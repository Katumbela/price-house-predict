Esta API utiliza vários modelos de machine learning para prever diferentes aspectos em diversos mercados.

Endpoints

1. Previsão de Preço de Casas

Utiliza um modelo de regressão linear para prever o preço de casas com base em características como área, número de quartos e número de banheiros.

Endpoint: /api/predict/house

Método: POST

Exemplo de Entrada:


{
    "area": 2000,
    "bedrooms": 4,
    "bathrooms": 3
}

Exemplo de Uso com curl:


curl -X POST https://price-house-predict.onrender.com//api/predict/house -H "Content-Type: application/json" -d '{
    "area": 2000,
    "bedrooms": 4,
    "bathrooms": 3
}'

Resposta:


{
    "predicted_price_usd": 400000.0,
    "predicted_price_aoa": 320000000.0
}

3. Previsão de Rotatividade de Funcionários

Utiliza um modelo de classificação para prever a probabilidade de rotatividade de funcionários com base em vários fatores.

Método: POST

Exemplo de Entrada:

    "last_evaluation": 0.7,
    "number_project": 3,
    "average_montly_hours": 150,

    "promotion_last_5years": 0,
    "sales": 2,
    "salary": 2
}

curl -X POST https://price-house-predict.onrender.com//api/predict/hr -H "Content-Type: application/json" -d '{
